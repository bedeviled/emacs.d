- here are my notes on trying to build jdee
- jdee - must first get elib from http://sourceforge.net/projects/jdee/files/jdee/Dependencies/elib-1.0.zip/download.  Then git submodule for jdee from http://github.com/emacsmirror/jdee.git.  Next, create a ~/.jdee-config-properties.  Mine looks like this
        #user editable build configuration parameters (this is generated by and overridden by /Users/mwall@texeltek.com/.jdee-config.properties when it exists)
        #Mon Sep 27 14:40:26 EDT 2010
        cedet.dir=/Applications/Emacs.app/Contents/Resources/lisp/cedet
        elib.dir=/Users/mwall@texeltek.com/.emacs.d/vendor/elib-1.0
        build.bin.emacs=/Applications/Emacs.app/Contents/MacOS/Emacs
        #config.time=2010/09/27 14\:40
        prefix.dir=/Users/mwall@texeltek.com/.emacs.d/vendor/jdee
Now get ant-contrib from http://sourceforge.net/projects/ant-contrib/files/ant-contrib/ant-contrib-0.6/ant-contrib-0.6-bin.zip/download and uzip.  Go to .emacs.d/vendor/jdee and export CLASSPATH=(location to ant-contrib-0.6.jar.  Now you should be able to run ant and build jdee. See java.el for more
