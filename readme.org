Vendor packages
- Install color-theme from
  http://download.savannah.gnu.org/releases/color-theme/.  Unzip and
  put color-theme.el in vendor and color-theme-example.el and
  color-theme-library.el in vendor/thems
- themes/twilight-emacs - theme used by defunkt -  git submodule from
  git://github.com/crafterm/twilight-emacs.git
- nav - Emacs Nav package - mercurial project on google code, hg clone
  https://emacs-nav.googlecode.com/hg/ emacs-nav, not committing .hg
  file, so you must clone it yourself
- magit - Magical Git for Emacs - git submodule from from
  http://github.com/philjackson/magit, say you need to install, but
  seems to work fine as is
- paredit - wget'd from http://mumble.net/~campbell/emacs/paredit.el,
  current version is 21
- slime - git submodule from git://git.boinkor.net/slime.git.  Worked
  first time adding to autoloads
- clojure-mode - gitsubmodule from
  http://github.com/technomancy/clojure-mode.git
- vivid chalk color theme - wget from
  http://github.com/evangineer/aquamacs-emacs-starter-kit/blob/master/site-lisp/vivid-chalk/vivid-chalk.el
- zenburn color theme - wget from http://www.emacswiki.org/emacs/download/zenburn.el
